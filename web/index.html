⏺ AskAnswer

基于 LangGraph 的命令行智能助手

按意图分流到「读本地文件 / 联网搜索 / 直接回答」，由 LLM 调度工具并整合结果，支持自评改写与 Markdown 渲染。

快速开始 GitHub

核心特性

🧭 意图分流

自动识别输入属于读文件 / 搜索 / 闲聊，分发到对应节点。

🔧 工具调度

天气、时间、计算、汇率、IP 查询等内置工具，模型按需调用。

🌐 MCP 热接入

SSE / streamable-HTTP / stdio 三种传输，运行时 /mcp <url> 即连。

🪞 自评改写

sorcery 节点评估答案质量，搜索路径允许改写关键词重跑一次。

🖥️ 类 Claude REPL

圆角欢迎框、箱式输入、⏺ Node(detail) 进度标记、Markdown 渲染。

⚡ Shell 前缀

!cmd 直接执行 shell 命令；命中高风险模式会二次确认。

安装 & 配置

pip install -e .

在项目根目录创建 .env：

OPENAI_API_KEY=...
TAVILY_API_KEY=...
OPENWEATHER_API_KEY=...   # 可选，启用天气工具

使用

askanswer "上海今天天气怎么样"   # 单次提问
askanswer                         # 交互模式
python -m askanswer               # 等价入口

斜线命令

命令说明
/help显示帮助
/clear清屏并开启新会话
/status查看当前会话信息
/mcp管理 MCP 服务
/exit / /quit / /q退出（也可 Ctrl-D）

工作流

START → understand ─┬─ file_read ───────────┐
                    ├─ answer (chat) ───────┤
                    └─ search → answer ─────┤
                         answer ⇄ tools     │
                                            ▼
                                         sorcery ─ {search 重搜 | END}
